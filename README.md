# Bank_Customer_Churn_Analysis with Power BI


## Overview
This Power BI report analyzes bank customer churn using data obtained from Kaggle's "Customer Churn Dataset" by Anand Shaw. The report aims to understand customer churn patterns and identify factors contributing to churn using visualizations and measures created in Power BI.

## Dataset
- **Dataset Link**: [Customer Churn Dataset](https://www.kaggle.com/datasets/anandshaw2001/customer-churn-dataset)
- **Churn_Modelling.csv**: This file contains information about bank customers, including their demographics, account details, and churn status.

## Power BI Report
The Bank Customer Churn Analysis report was created using Power BI. It utilizes Power Query Editor to transform and load data from the provided dataset. Additionally, various measures and visualizations were created to analyze and visualize the churn data.

## Files
- **Churn_Modelling.csv**: Contains the dataset with bank customer information.
- **Bank Customer Churn Analysis.pbix**: This Power BI file contains the dashboard and visualizations created for analyzing customer churn.
- **Bank Customer Churn Analysis.png**: An image file showcasing the dashboard for quick reference.

## Visualizations
The report includes various types of visualizations to analyze customer churn, such as:
- **Cards**: Displaying key metrics like total customers, churn rate, etc.
- **Stacked Bar Chart**: Showing churn status based on different demographics or account attributes.
- **Stacked Column Chart**: Analyzing churn trends over time or across different segments.
- **Area Chart**: Visualizing churn trends over a period.
- **Donut Chart**: Illustrating the distribution of churn status.
- **Map**: Geospatial visualization of churn distribution.
- **Slicers**: Interactive filters for exploring data by different dimensions.

## Usage
1. Ensure you have Power BI Desktop installed on your system.
2. Download the provided dataset file (`Churn_Modelling.csv`).
3. Open the `Bank Customer Churn Analysis.pbix` file in Power BI Desktop.
4. Use Power Query Editor to load and transform the dataset as needed.
5. Explore the visualizations and measures in the report to gain insights into bank customer churn patterns.
